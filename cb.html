CB-Studio

Design Engineer

CB-Studio was a design agency that i formed together with a colleague of mine from architecture school. At the start, we focused solely on interior design projects. Later on, we worked on a wide array of design projects - from web development to brand identity.

One of the aspects that we considered essential in this project was to have a website which is intuitive and easy to use. Moreover, the interoperability between different platforms was also key. We managed to entertain these points and create a website that adapts to each device ( having mobile, tablet and desktop support).

When making the website, I learned lots about CSS and HTML. The Squarespace CMS that we use was indeed helpful but I had to use lots of coding to get the results that we both wanted. First of all, making the website bilingual was quite a challenge. Changing the default fonts and the way they behave in the paragraphs. Moving different elements while maintaining the interoperability between the various platforms. Having the website adapt to all the changes that we made on the desktop version on the mobile version as well. Introducing a javascript that animates the reveal of the paragraphs and text and many more!

Regarding CB-Studio, we had a great run but we found it hard to get into the external design market. That was our goal so we chose to end the journey there. It has been an amazing experience for both of us. We learned key things about running a business, working with clients and marketing/growing your own entrepreneurial pursuit.
